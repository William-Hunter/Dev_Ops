# Dev_Ops
做运维写的一些自动化任务脚本


### autoPullbackup.sh
-------------
自动拉取备份文件，设置成定时任务

### backupMysql.sh
-------------
备份数据库,-R参数到处存储过程


### backWebapp.sh
-------------
压缩备份web应用和数据库


### compackLog.sh
-------------
定时清理累计的tomcat的日志，


### tomcatMonitor.sh
-------------
监控常驻端口的tomcat进程是否在运行中，如果不在，就启动tomcat，以此保持tomcat实例故障恢复
